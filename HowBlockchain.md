# 如何導入區塊鏈
- Wallet: 區塊鏈的用戶帳號
- 架設鏈: Parity
- 與鏈溝通: REST API
- Smart Contract: https://cryptozombies.io/
- Backend: TideDog

## Wallet
* BIP32 - Hierarchical Deterministic wallet
* BIP39 – 助記辭
* BIP44 – 多幣種/帳戶支援
